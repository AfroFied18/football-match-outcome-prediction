# football-match-outcome-prediction
Machine learning project predicting football match outcomes using European Soccer Database.

## Dataset

This project uses the European Soccer Database.

## ⚠️ Note
Dataset is large and excluded from the repository. Please download manually from the link below.

Dataset source:
https://www.kaggle.com/datasets/hugomathien/soccer

Download the dataset and place `database.sqlite` in the project folder before running the notebook.
